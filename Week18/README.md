# 18주차: 데이터베이스 연동 기초

"서버 껐다 켜도 데이터가 남아있어야죠!"
게임 서버에서 DB는 필수입니다. 이번 주에는 DB 연동의 개념을 익힙니다.

## 1. 핵심 개념
- **ODBC**: 윈도우 표준 DB 인터페이스.
- **Connection Pool**: DB 연결을 미리 맺어두고 재사용하는 기술 (쓰레드 풀과 비슷).
- **SQL Injection**: 해킹 방지를 위해 Prepared Statement 사용 필수.

## 2. 실습 가이드
- 실제 DB(MySQL 등)를 설치하는 건 너무 복잡하므로, 여기서는 **Mock DB(가짜 DB)** 클래스를 만들어 비동기 DB 처리 패턴을 익힙니다.
1. **01_mock_db.cpp**: DB 쿼리를 요청하고 콜백으로 결과를 받는 패턴.

## 3. 빌드 및 실행

.\build_cmake.bat
